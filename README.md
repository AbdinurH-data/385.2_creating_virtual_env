# Creating virtual ENV (env)
  -- 1. create your project folder and make sure you terminal is at the root of the project
  -- 2. To create virtual env, run in terminal `python -m venv venvname`

# Using a virtual env
   -- if you don't activate yor virtual env, when you install packages they won't be any specific 
   -- To activate windows: `venvname/Scripts/activate`
   -- To deactivate `deactivate`
